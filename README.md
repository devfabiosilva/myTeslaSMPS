# myTeslaSMPS

High Voltage Range Tesla Switching Mode Power Supply for IoT and Embedded Systems

## Brief description

This WVR-SMPS (Wide Voltage Range Switching Mode Power Supply) or simply Tesla SMPS is a wide range and high voltage input for low power embedded systems. This device allows input voltage as low as 100 VAC and as high as 820 VAC in wide frequency range (45 − 450 Hz) or even in DC voltage as low as 142 VDC and as high as 1,160 VDC

### Features

- Supply for low power Embedded System (max. power 20W)
- Safe high insulation for output voltage (max. 2,600V)
- Limited inrush current 42.6 A @ 1,160 Volts DC (0.38 A^2.s)
- High frequency range in AC operation (45 − 450Hz)
- AC/DC operation
- Static short circuit/overload protection
- Under voltage protection
- Low cost components (only 3 active elements!)
- Startup circuit
- Limited ID (current drain) monitor
- Secondary side grounded for security

## Specifications

### Input Voltage:

AC 100 ~ 820 Volts (45 - 450 Hz)
